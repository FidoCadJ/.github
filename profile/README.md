# FidoCadJ

## Introduction

FidoCadJ is an easy to use 2D vector drawing program, with a vast library of electrical symbols and footprints.
Its drawings can be exported in several graphic formats (PDF, EPS, PGF for LaTeX, SVG, PNG, JPG). 
FidoCadJ can be useful as a basic schematic capture for rapidly obtaining a drawing that can be put in a presentation or in a paper.
FidoCadJ employs a file format containing text, very compact and perfect for copying and pasting from/into newsgroups and forum messages. 

## How to get involved

You are welcome to participate to the development of FidoCadJ.  
If you spot a typo, an error in the manual or in the program, any correction is very welcome. 
We will aleo be very happy to know what do you think about FidoCadJ.
We would be glad to receive some help: if you know Java and you want to work on the source code you will be welcome. 
If you are not fluent with Java coding, you can participate to the project by improving the manuals or translating the interface, or just giving an advice.
